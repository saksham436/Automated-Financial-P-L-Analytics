# Financial Analytics Dashboard

## Project Overview

This project focuses on building an interactive Financial Analytics Dashboard using Excel, Power Query, SQL, Power Pivot, Advanced Financial Modeling, and DAX Analytics. The objective of the project was to transform raw accounting and transactional data into meaningful financial insights that support faster and smarter business decision-making.

The project involved handling complex, multi-department financial ledgers containing inconsistent expense structures and unorganized accounting records. Using Power Query, efficient ETL pipelines were created to clean, normalize, and automate data preprocessing tasks. This significantly reduced manual cleaning effort by more than four hours while improving data consistency and reporting accuracy.

A centralized data model was developed using Power Pivot to establish relationships between multiple financial datasets. Advanced DAX measures were engineered to dynamically calculate important KPIs such as revenue, operational costs, profitability, departmental expenses, and variance analysis. These measures enabled detailed cost-center evaluation and helped track organizational financial performance efficiently.

The dashboard also focused heavily on trend analysis and expense tracking. Cash-flow patterns, operational spending, and revenue streams were analyzed over time to identify hidden cost drivers and detect unusual business fluctuations. The project successfully highlighted major financial variances across core operational categories, helping uncover actionable business insights.

Interactive visualizations and dynamic reporting features were implemented to make the dashboard recruiter-ready and business-friendly. The final solution transformed raw accounting data into structured visual intelligence that improved financial reporting efficiency and accelerated data-driven strategic planning.

This project demonstrates strong skills in:

* Data Cleaning & Transformation
* Financial Modeling
* Power Query Automation
* DAX Analytics
* SQL-Based Data Handling
* Dashboard Development
* Business Intelligence & Reporting

The dashboard showcases how modern analytics tools can simplify financial analysis and improve organizational decision-making through automation and visualization.
